两个有序数组，求中位数问题

方法1:

把a，b两个有序数组合并成一个新的有序数组c，然后求c数组的中位数。

时间复杂度：排序算法的时间复杂度

方法2:

方法1的优化

假设a数组的长度是m，b数组的长度是n

(1)计算索引的位置

如果(m+n)是基数：那么中位数的索引i是(m+n-1)/2，中位数是a[i]

如果(m+n)是偶数：那么中位数的索引i是(m+n)/2，中位数是(a[i-1]+a[i])/2

(2)根据计算出来的索引i的位置，从a，b数组中比较找到第i大的数

时间复杂度是(m+n)

方法3：

时间复杂度为log(m+n)

注意：

1 假设数组a是基数

假设数组a的长度是n，索引是i，中位数是a[i]，则需要满足以下条件

  1. 数组左边的个数+1 = 数组右边的个数， 所以a[0,i) + 1 = a[i, n)

2 假设数组a是偶数

假设数组a的长度是n，索引是i，中位数是(a[i-1] +a[i])/2，则需要满足以下条件

  1. 数组左边的个数 = 数组右边的个数， 所以a[0,i) = a[i, n)

3 两个数组的和如果是基数

假设数组a的长度是m，数组b的长度是n；我们假定把a，b两个数组合成一个数组c，长度是(m+n)

假设数组a索引i，数组b索引j；把数组c分成两部分; c的左边a[0,i) + b[0,j)，c的右边a[i,m) + b[j,n)

则中位数min(a[i], b[j])，需要满足以下条件

  1. 数组c右边的数小于数组c左边的数，所以a[i-1] < b[j] && b[j-1] < a[i]

  2. 数组c右边的个数+1 = 数组左边的个数，所以 [0,i) + [0,j) + 1 = [i,n) + [j, m)

4 两个数组的和如果是偶数

假设数组a的长度是m，数组b的长度是n；我们假定把a，b两个数组合成一个数组c，长度是(m+n)

假设数组a索引i，数组b索引j；把数组c分成两部分; c的左边a[0,i) + b[0,j)，c的右边a[i,m) + b[j,n)

则中位数( max(a[i-1], b[j-1]) + min(a[i],b[j]) )/ 2需要满足以下条件

  1. 数组c右边的数小于数组左边的数，所以 a[i-1] < b[j] && b[j-1] < a[i]

  2. 数组c右边的个数 = 数组左边的个数，所以 [0,i) + [0,j) = [i,n) + [j, m)

算法实现

我们可以用二分法，找到数组a的索引i和数组b的索引j，满足 a[i-1] < b[j] && b[j-1] < a[i]。

通过分析我们知道i 和j的关系是j = (m+n)/2 - i；所以我们可以得到以下算法

计算左边的最大值和右边的最小值，有一些边界条件需要注意
